#### Matches the following

```text
// BUILD_INCLUDE(filePath)
// BUILD_INCLUDE(filePath)[options]
// BUILD_INCLUDE(filePath)
[options]

* BUILD_INCLUDE(filePath)
* BUILD_INCLUDE(filePath)[options]
* BUILD_INCLUDE(filePath)
[options]

/* BUILD_INCLUDE(filePath)
/* BUILD_INCLUDE(filePath)[options]
/* BUILD_INCLUDE(filePath)
[options]

<!-- BUILD_INCLUDE(filePath)
<!--BUILD_INCLUDE(filePath)-->
<!--BUILD_INCLUDE(filePath)[options]-->
<!--BUILD_INCLUDE(filePath)
[options]-->
```
